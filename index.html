Components of the web
  • HTML - Main type of document on the web
  • HTTP - the main protocol of the web

Tag: An HTML tag is always contained within angled brackets. Most tags have an opening tag (< p> for example) and a closing tag, (< /p>). Some tags (called "void" tags) do not require a closing tag (like the < br> tag).

  • < em> - italics
  • < a> - Anchor - for making links
  • < image> - void tag - has no content. Images have no content, they do not need a closing tag.
  • < br> - also a void tag
  • anchor tags use the href attribute while img tags use the src attribute.
  • < br> - “inline”. ends a line. just text
  • < p> - block. Makes a box around the text. Can have width and height.
  • container elements: elements that do nothing but contain stuff inside themselves.
  • < span> - inline
  • < div> - block
  • both contain content

Element: An HTML element refers to everything within a set of opening and closing tags.

Attribute: This is a property of an HTML element. For example, to set the href attribute of an anchor tag to the Udacity URL, you would write < a href="www.udacity.com">
